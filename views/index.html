End Points


Product Table

GET /products Show all products
GET /products/:id Show product
POST /products Add product
PUT /products/:id Update products
DELETE /products/:id Deletes product

User Table

GET /users Show all users
GET /users/:id Show user
POST /users Add user
DELETE /users/:id Deletes user

Cart Table

GET /users/:id/cart Shows all items in the cart of the user with the specific id
GET /users/:id/cart/:cartId Gets a specific item from the cart of the user with the specific id
POST /users/:id/cart Updates the items in the cart of the user with the specific id
PUT /users/:id/cart Updates the items in the cart of the user with the specific id
DELETE /users/:id/cart Clears the cart of the user with the specific id
